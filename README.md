# X ツール（いいね・ブックマーク自動化）

X（旧Twitter）のタイムライン上で、ツイートの**いいね**や**ブックマーク**を一括で実行・解除するブックマークレットです。

## 機能

- **いいね** … 未いいねのツイートにいいねを付ける
- **いいね解除** … いいね済みのツイートのいいねを外す
- **ブックマーク** … 未ブックマークのツイートをブックマークする
- **ブックマーク解除** … ブックマーク済みのツイートのブックマークを外す
- **いいね＆ブックマーク（両方ON）** … いいねとブックマークを同時に付ける
- **いいね＆ブックマーク（両方OFF）** … いいねとブックマークを同時に外す

## 使い方

### 1. ブックマークレットの登録

1. `auto_like_bookmark.js` の**全文**をコピーする
2. ブラウザのブックマークバーに「新しいブックマーク」を追加
3. 名前は任意（例：`X いいね・ブックマーク`）
4. URL の欄に `javascript:` の後に、コピーしたコードをそのまま貼り付けて保存

### 2. 実行手順

1. [X（Twitter）](https://x.com) にログインし、タイムラインや検索結果など、操作したいツイートが表示されているページを開く
2. 登録したブックマークレットをクリック
3. 表示されたダイアログで以下を設定して「開始」を押す
   - **操作** … いいね / いいね解除 / ブックマーク / ブックマーク解除 / 両方ON / 両方OFF
   - **開始時期（任意）** … この日付以降のツイートのみ対象にする
   - **終了時期（任意）** … この日付以前のツイートのみ対象にする
   - **待機時間（秒）** … 1件ごとの待ち時間（デフォルト 3 秒）

### 3. 実行中

- 画面右下に「実行状況」が表示されます
- 日付フィルターを使わない場合は、デフォルトで 5 件まで処理したら終了します
- 日付を指定した場合は、その期間内のツイートを順に処理し、該当がなくなったら終了します
- ページが再読み込みされても、約 3 秒待ってから自動で再開を試みます

## 注意事項

- **X の利用規約**に従ってご利用ください。過度な自動操作は制限やアカウント停止の原因になる可能性があります
- 待機時間は短くしすぎず、**3 秒以上**を推奨します
- X の画面仕様が変わると、ボタンのラベルや構造が変わり、正しく動かなくなる場合があります
- このツールの利用により生じた不具合や損害について、作者は責任を負いません
