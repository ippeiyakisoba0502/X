# X ツール（ブックマークレット）

X（旧Twitter）のタイムライン上で、**いいね・ブックマーク**の一括操作や**ポストの通報**を自動で行うブックマークレットです。

---

## 1. いいね・ブックマーク自動化

ツイートのいいねやブックマークを一括で実行・解除します。

### 機能

- **いいね** … 未いいねのツイートにいいねを付ける
- **いいね解除** … いいね済みのツイートのいいねを外す
- **ブックマーク** … 未ブックマークのツイートをブックマークする
- **ブックマーク解除** … ブックマーク済みのツイートのブックマークを外す
- **いいね＆ブックマーク（両方ON）** … いいねとブックマークを同時に付ける
- **いいね＆ブックマーク（両方OFF）** … いいねとブックマークを同時に外す

### 使い方

1. `auto_like_bookmark.js` の**全文**をコピーする
2. ブラウザのブックマークバーに「新しいブックマーク」を追加し、URL に `javascript:` の後にコードを貼り付けて保存
3. [X](https://x.com) のタイムラインや検索結果を開き、ブックマークレットをクリック
4. ダイアログで**操作**・**開始/終了時期（任意）**・**待機時間（秒）**（デフォルト 3 秒）を設定して「開始」

実行中は画面右下に「実行状況」が表示されます。日付を指定しない場合は 5 件で終了、指定した場合はその期間内のツイートを処理します。

---

## 2. 通報ツール（X通報ツール）

タイムライン上のポストを「**ヘイト、攻撃的な行為、または嫌がらせ**」として一括で通報します。

### 使い方

1. `auto_alert_spam.min.js` の**全文**をコピーする（1行になっているためブックマークレット向け）
2. ブラウザのブックマークバーに「新しいブックマーク」を追加し、URL に `javascript:` の後にコードを貼り付けて保存
3. [X](https://x.com) のタイムラインを開き、ブックマークレットをクリック
4. ダイアログで以下を設定して「開始」を押す
  - **開始日（任意）** … この日付以降のツイートのみ対象
  - **終了日（任意）** … この日付以前のツイートのみ対象
  - **待機時間（秒）** … 1件ごとの待ち時間（デフォルト **15 秒**）

実行中は画面右下に「実行状況」が表示されます。対象期間内のツイートを上から順に処理し、該当がなくなったら終了します。通報は無制限で行われます。

### 補足

- X の通報フロー（「ポストを報告」→ 理由選択 → 次へ → 完了）を自動で操作します
- 待機時間を短くしすぎると、X の内部API（`report/flow.json`）がレート制限でエラーになることがあります。**15 秒以上**を推奨します

---

## 注意事項

- **X の利用規約**に従ってご利用ください。過度な自動操作は制限やアカウント停止の原因になる可能性があります
- いいね・ブックマークツールは待機時間 **3 秒以上**、通報ツールは **15 秒以上**を推奨します
- X の画面仕様が変わると、ボタンやモーダルの構造が変わり、正しく動かなくなる場合があります
- このツールの利用により生じた不具合や損害について、作者は責任を負いません

## 動作環境

- X（旧Twitter）のウェブ版（[https://x.com）で動作を想定しています](https://x.com）で動作を想定しています)
- モダンなブラウザ（Chrome、Firefox、Safari、Edge など）で利用できます
